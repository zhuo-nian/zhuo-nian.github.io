发表于2025-09-28
首先要区分jdk和jre：JRE (Java Runtime Environment)：是 运行 Java 程序的环境。JDK (Java Development Kit)：是 开发 Java 程序的工具包，它包含了 JRE。 截止到博客写作时间：Java最新版本为25 下载地址：https://www.oracle.com/cn/java/technologies/downloads/作者的环境路径： D:\Java\jdk-21 配置系统环境变量： 创建一个名为JAVA_HOME的变量，值为jdk的安装路径，作者就是D:\Java\jdk-21 在“系统变量”中，找到名为 Path 的变量，选中它，点击 编辑（Edit）,点击“新建”，然后添加： %JAVA_HOME%\bin 验证：java -version 和 javac -version
